# i want a name · 秒出好域名 ⚡

> 中文 | [EN](README.md)

**有了好点子，却被域名卡住？** 告诉 AI 你的想法，它会瞬间给出好记又有品牌感的域名建议。实时查询可用性，一键跳转注册商。完全在你的浏览器运行——数据不离开你的设备。

---

## 我们都遇到过的痛点

这种感觉你肯定懂：脑子里有个绝妙的产品创意，但一到起域名就犯难...

- 想到的好域名全被注册了 😫
- 花了几个小时头脑风暴，结果一查全不可用
- 还能注册的？要么太长，要么太怪，要么就是不够酷
- 一个个手动查可用性，慢得让人抓狂

**这个工具就是来终结这种折磨的。**

---

## 它的特别之处

### 🧠 **AI 创意引擎**
用大白话描述你的想法（中文英文都行！），就能得到一串精选的短小精悍、朗朗上口的品牌域名。再也不用对着空白页苦思冥想了。

### ⚡ **秒查可用性**
别再一个个手动检查了。我们自动对每个建议运行 WHOIS 查询，直接告诉你哪些可以注册，哪些已经被谁抢走了。

### 🔗 **一键直达注册**
看中了某个域名？点一下就跳到你常用的注册商页面，域名已经帮你填好。GoDaddy、Namecheap、Cloudflare——随你挑。

### 💬 **多项目管理**
同时搞好几个项目？给每个创意开个独立对话，重命名它们，随时切换。你的脑暴历史井井有条。

### 💾 **数据由你掌控**
导出对话为 JSON 文件——备份、分享给团队、或者以后导入回来。你的数据，你说了算。

### 🔒 **真正的隐私保护**
- API 密钥？存在你的浏览器里
- 聊天记录？留在你的设备上
- 没有追踪，没有分析，没有云端存储（除非你自己部署）
- 只连接你选择的 AI 服务商和 WHOIS 服务

### 🎨 **随心定制**
- 深浅色主题，保护你的眼睛
- 中英文界面切换
- 自定义系统提示词，按你的方式引导 AI
- 兼容 OpenAI 接口（可用 GPT、Azure 或任何兼容服务）

## 30 秒上手

```bash
# 克隆项目本地运行
npm install
npm run dev
# 打开 http://localhost:3000 开始头脑风暴！
```

**就这么简单。** 不用配置数据库，不用折腾环境变量（除非你要部署），装完就能跑。

## 随处部署

### Cloudflare Pages（推荐）
```bash
npm install && npm run build
```
- Node 版本：20
- 框架：Next.js（自动识别）
- 输出目录：`.next` 或使用 `@cloudflare/next-on-pages`
- 费用：大多数场景下**完全免费**

因为所有逻辑都在客户端运行，你可以部署到任何静态托管平台：Vercel、Netlify、GitHub Pages，甚至自己的服务器。

## 安全第一

你的 API 密钥就是钱包的钥匙，我们懂。

- **不离开浏览器**：存在 `localStorage` 里，绝不发往任何后端
- **导出文件包含一切**：如果你导出对话，那个 JSON 里有完整历史——请妥善保管
- **开源透明**：代码全在这儿，随便看，随便 fork，随便审计
